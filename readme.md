# Flappy-bird-mina
微信小游戏版flappy-bird
因为没有相关资质，所以只能发布一个在线的H5 demo版本
master分支为可以直接运行在微信开发工具中
html分支为H5版本
两者差异只为canvas和image创建方式不同

| H5版本      | 微信小游戏 |
| :-------- | --: |
|  new Image()  |  wx.createImage()   |
| this.canvas.getContext('2d')  |  wx.createCanvas()  |
